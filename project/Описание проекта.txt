Описание проекта.
В проекта предствлено мое понимание работы базы данных сайта Кинопоиск.
Файл Kinipoisk_tables создает саму базу данных состоящую из нескольких таблиц: таких как Пользователи,Профиль,Запрос на дружбу,
Название фильма, Оценка фильма и тд. Описание каждой из таблиц присутсвует в теле файла.

Для демонстарации работы база данных была заполнена Dummy даннымы с помощью http://filldb.info, некоторые правда пришлось редактировать руками.
